The User Login program demonstrates simple user authentication functions to ensure that the right user is
given the appropriate access to different data. This is an attempt to ensure Confidentiality in the CIA Triad.
Confidentiality is breached when any user that logs into a system has access to any and all info stored
within the system, regardless of level of access. This can hurt the second part of the triad, Integrity.
If any user can manipulate all data in a system this could lead to the damaging of much the date stored
within the system.
